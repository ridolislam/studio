# **App Name**: LeadPulse

## Core Features:

- Multi-format File Importer: Seamlessly upload and process data from .xlsx, .xls, .csv, and .txt file formats for centralized lead processing.
- AI Data Parsing Tool: A generative logic tool that automatically recognizes and extracts Source Links, Phone Numbers, and Business Names from unstructured files.
- Live Usage Analytics: A real-time dashboard displaying key indexing, usage limits (0-100 scale), and total lead counts.
- Active Progress Monitor: Visual tracking of data processing with live success/failure logs and a dynamic progress bar.
- Categorized Result Boxes: Smart displays for Mobile, Landline, and Invalid results that feature instant counter updates and one-click download to Excel.
- Sticky-Header Smart Filter: A high-performance table view that organizes valid and segregated phone data with a sticky header for better navigation.
- Session Process Control: Global 'Start' and 'Stop' buttons to manage data extraction workflows in real-time.

## Style Guidelines:

- Dark-themed palette anchored in precision. Background: Deep Indigo-Black (#0e0d14); Primary: Electric Violet (#7155ff) for visual punch; Accent: Cyber Blue (#1a71ff) for highlighting interactive states.
- Pairing 'Inter' for clean, neutral headlines and body text with 'Source Code Pro' for numerical displays, key indices, and dashboard data counters.
- Outline-style SVG icons using the Electric Violet color for file types and operational controls (Play, Stop, Download).
- Compact, high-density dashboard layout with sticky-header tables and grid-based status cards for optimal visibility on all screen sizes.
- Pulse effects on progress bars and subtle slide-in transitions when result boxes update their internal counters.
